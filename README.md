This project aims to conduct an EDA on NVIDIA's historical stock price data. I downloaded the CSV data directly from Nasdaq (https://www.nasdaq.com/market-activity/stocks/nvda) and wrote custom code to analyze the data, plot a long-term stock price trend line, and generate statistics analysis to observe the company's long-term market trajectory. During the development process, I utilized Gemini to research and learn which specific functions should be applied when writing the code and also submitted the code to Gemini for review and correction.

I imported core Python data science libraries, including Pandas for tabular data manipulation, NumPy for mathematical matrix operations, and Matplotlib for data plotting and visualization. Concurrently, a built-in try-except exception handling mechanism was implemented within the program to ensure that the system preemptively prompts error messages when encountering abnormal file names or paths, preventing the script from abruptly crashing. Since the closing prices in the raw fields contained dollar signs and commas, the program utilizes regular expressions to strip away these non-numeric symbols and forces a type conversion into floats, thereby enabling computational functionality. The date field was also parsed into a standardized datetime format, and the entire dataset was chronologically reordered. Also, the program utilizes NumPy's polyfit algorithm to perform linear regression and establish a long-term baseline. By letting the computer automatically execute a first-order linear regression, it calculates the optimal slope and intercept of this stock price dataset, thereby constructing a mathematical model for the overall long-term trend line.

In the generated statistical chart, the blue line illustrates NVIDIA's stock price fluctuations, while the red line represents the long-term trend line. A descriptive statistics table is automatically output beneath the chart, comprising the total number of trading days, average price, standard deviation, maximum price, and minimum price. Although NVIDIA's stock price exhibits sharp short-term volatility driven by market sentiment and industry hype, the red linear trend line demonstrates a distinct upward slope toward the upper right. The overall economic fundamentals and long-term momentum are both firmly situated in a phase of robust, upward growth.
